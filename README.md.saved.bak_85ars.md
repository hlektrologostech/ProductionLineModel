#  Μοντέλο γραμμής παραγωγής

**Σχολείο:** ΕΠΑΛ Θήβας.

**Τίτλος έργου:** «Χρήση της τεχνητής νοημοσύνης σε βιομηχανικές γραμμές παραγωγής »
